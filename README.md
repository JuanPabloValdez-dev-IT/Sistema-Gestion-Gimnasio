# 🏋️‍♂️ Gym Management System (CLI)

A comprehensive console-based Gym Management System developed in C. This project demonstrates core backend logic, data persistence, and dynamic memory allocation applied to a real-world business scenario.

## 🚀 Key Features
* **User Authentication:** Secure login system with strict credential validation (uppercase checks, consecutive numbers restrictions, etc.).
* **CRUD Operations:** Complete management of gym members (Create, Read, Update, and Logical/Physical Delete).
* **Data Persistence:** Secure data storage and retrieval using binary files (`.dat`).
* **Dynamic Memory:** Implementation of linked lists using pointers for dynamic data handling.
* **Real-time Statistics:** Analytics generation based on active members and membership plans.

## 🛠️ Technologies & Skills Applied
* Language: **C / C++**
* Pointers & Dynamic Memory Allocation (`malloc`, `free`)
* Complex Data Structures (Linked Lists)
* Binary File Handling
* Business Logic Implementation

## 📌 About this Project
This software was developed as a final assignment for the *Algorithms and Data Structures* course. It applies theoretical programming concepts to solve actual business needs, showcasing the ability to translate business requirements into functional, secure backend code.
